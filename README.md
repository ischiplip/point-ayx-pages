# point-ayx-pages

本仓库用于归档与发布多个独立的 HTML 页面。这些页面各自独立运行，不依赖于特定的域名或外部服务。

## 项目简介

`point-ayx-pages` 是一个静态页面集合仓库，主要用于存放各类 HTML 页面文件。每个页面均为独立实现，可直接在浏览器中打开使用。仓库本身不提供任何动态功能或后端服务。

## 目录说明

```
point-ayx-pages/
├── pages/         # 存放所有 HTML 页面源文件
├── assets/        # 页面引用的资源文件（CSS、JS、图片等）
└── README.md      # 本说明文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名清晰，便于识别。
- **assets/**：公共资源文件，按类型存放，供各页面引用。

## 页面归档说明

- 所有页面均以 HTML 格式保存，可在本地直接打开。
- 页面之间无耦合，各自独立维护。
- 部分页面可能引用 `assets/` 下的公共资源，请确保路径正确。
- 页面内容不包含任何第三方跟踪代码或外部请求，完全自包含。

## 维护说明

- 新增页面：在 `pages/` 下创建新文件或子目录，并更新本 README 的页面列表（如有需要）。
- 更新页面：直接修改对应 HTML 文件即可。
- 资源更新：添加或修改 `assets/` 下的文件时，注意检查引用路径。
- 提交前请确保页面在本地浏览器中能正常显示。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/point-ayx-pages.git
   ```
2. 进入 `pages/` 目录，双击任意 HTML 文件即可在浏览器中查看。
